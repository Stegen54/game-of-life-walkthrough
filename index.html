Conway's Game of Life

This is a cellular automaton simulation where cells live or die based on simple rules:

Rules:

  1. Any live cell with fewer than 2 live neighbors dies (underpopulation)
  2. Any live cell with 2 or 3 live neighbors lives
  3. Any live cell with more than 3 live neighbors dies (overpopulation)
  4. Any dead cell with exactly 3 live neighbors becomes alive

Controls:

  • Start/Stop: Toggle the simulation
  • Reset: Generate a new random pattern
